## 🗺️ 场景全览

### 🟣 车内体验 · In-Cabin（6 个场景）

| 场景 | 核心价值 | 关键指标 |
|------|---------|---------|
| **座舱助手** | 多模态感知 + 情绪识别，从语音工具升级为行车伴侣 | 意图准确率 >92%，端到端响应 <800ms |
| **智能导航** | 融合路况、能耗、偏好，动态规划最优行程 | ETA 误差 <3 分钟，能耗预测误差 <5% |
| **驾驶安全** | 疲劳检测、危险预警、紧急事件主动介入 | 疲劳识别准确率 >88%，误报率 <5% |
| **健康出行** | 车内空气质量、驾驶员生理监测、长途健康管理 | 疲劳干预接受率 >70% |
| **试驾体验** | 个性化路线、实时数据可视化、试驾后报告 | 试驾→洽谈转化率提升 >15% |
| **智能停车** | 车位预订、无感支付、场内导航、动态定价 | 找车位时间减少 >50% |

| 场景 | 核心价值 | 关键指标 |
|------|---------|---------|

| **售后服务** | 故障诊断、维保预约、工单全流程自动化 | 自助解决率 >50%，响应 <30 秒 |

## 🔗 Agent 协同数据链

28 个场景通过 **7 条主要协同链路** 形成有机整体，而非孤立工具：

```
链路一 · 用户全旅程链
舆情口碑 → 展厅线索培育 → 试驾体验 → 智能销售 → 用户生命周期运营 → 二手车评估

链路二 · 车辆安全与风险管控链
驾驶安全 → 健康出行 → 保险理赔 → 召回管理 → 合规数据治理

链路三 · 研发制造智能链（飞轮）
ADAS数据闭环 → 研发制造 → 召回管理 → 售后服务 → ADAS数据闭环

链路四 · 能源与补能生态链
能源管理 → 充电补能 → 智能导航 → 座舱助手 → 出行服务

链路五 · 渠道运营与定价优化链
车队管理 → 智能定价促销 → 经销商运营 → 展厅线索培育 → 用户生命周期运营

链路六 · 企业内部智能运营链
供应链协同 → 财务智能 → 智能定价促销 → HR人才管理 → 研发制造

链路七 · 出海全链路合规链
供应链协同 → 能源管理 → 出海合规 → 合规数据治理 → ADAS数据闭环
```

## 💡 设计原则

文档中所有场景设计遵循以下原则：

**1. AI 辅助，人工决策**
安全件质检、保险理赔拒赔、召回启动、信贷审批等高风险决策，Agent 只提供辅助，最终决策权保留给人。

**2. 落地优先于愿景**
每个场景都有清晰的 Phase 1（0-3/4 个月）MVP 范围，优先选择技术成熟度高、数据基础好、用户价值明确的切入点。

**3. ROI 可量化**
ToB 场景给出年度价值测算，ToC 场景给出用户体验提升的量化指标，每个场景都有明确的商业价值支撑。

**4. 合规内嵌而非附加**
每个场景的风险约束模块覆盖《个人信息保护法》《汽车数据安全管理规定》《数据安全法》等核心法规，合规是设计的一部分，不是事后补丁。
